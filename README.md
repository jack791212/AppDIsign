# 🏹 暗影獵手（Shadow Hunter ARPG）

一個手機**直版（縱向）動作 RPG**，純 HTML5 + Canvas，手機瀏覽器打開即可遊玩。
操控承襲《弓箭傳說》（拖曳移動），但採 ARPG 式**持續自動攻擊**、成長系統為**暗黑破壞神式的永久養成**：
角色不會重置，靠**刷怪、掉裝、Build** 不斷變強。

## 核心玩法

- **拖曳螢幕移動**角色（虛擬搖桿）
- **持續自動攻擊**：自動朝最近敵人射箭（移動中也會射），專注走位閃避
- **開場劇情**：首次進入會有電影運鏡，村長跑來託付一把破舊短弓（劇情對話系統）
- 異常狀態會在角色頭上以文字顯示（🔥燃燒／❄️緩速／⚡麻痺）
- 城鎮鐵匠/女神各有房屋，站在自家門口；鐵匠強化有鐵鎚動畫與成功率
- （主動技能目前暫時關閉）
- **電腦快捷鍵**：B 背包、C 天賦、T 一鍵回城（受傷中斷）、空白鍵衝刺、Q 大招
- **音效**：Web Audio 即時合成（射擊/命中/暴擊/受傷/升級/Boss 大招…），右上 🔊 可靜音
- **死亡懲罰**：損失 20% 金幣（等級/裝備/戰利品保留）
- **長期目標**：每隻 Boss **首殺**給額外金幣＋保證傳奇；累積的 Boss 戰利品提供**永久加成**（每隻 +3% 攻擊、+20 生命）
- 回城後城鎮會出現**回歸傳送門**可傳回原本的區域；靠近🔨**鐵匠 NPC**（頭上按鈕）開啟**對話**，可選強化或賭裝
- **裝備強化**：選裝備 → 預覽強化後數值與**成功率** → 確認；+5 以上可能失敗（−1 級），+8 以上失敗會**爆炸消失**
- **敵人元素**：火（燃燒）、冰（減速）、雷（**麻痺**，移動會突然頓一下）；對應區域主題
- 玩家身側會顯示**裝備中的武器**，其朝向即為攻擊面向
- 對話劇情系統（可用於 NPC 交談/交易、未來 Boss 開場）
- 一般怪物會先在地上閃紅色預警約 1 秒才出現
- **連續擊殺**：3 秒內持續擊殺累積連殺數，數字會震動（越多越劇烈），中斷則重置
- 金幣與經驗都改為**地上掉落物**（🪙 金幣、🧋/🥟 經驗球），靠拾取範圍自動吸取
- 手機操作為**畫面下方 15% 的固定搖桿**（實心控制區）；血量顯示在角色下方、經驗條在控制區上緣
- 城鎮有 🔨 鐵匠（強化/賭裝）與 🧚 女神（回復生命、鼓勵）
- 在開放區域**自由刷怪練等**，怪物會持續重生
- 擊殺掉落**裝備**，走過去拾取
- 透過**傳送門**前往更高等級的區域（部分需達到等級才解鎖）
- **角色永久成長，死亡不會重置**（自動存檔於瀏覽器 localStorage）

## 三大成長系統

### 1. 等級
每擊殺怪物獲得經驗，升級提升基礎攻擊與生命，並獲得 **1 點天賦點**。

### 1.5 功法（🌀 按鈕）
三條平行：火🔥(傷害/燃燒)、雷⚡(連鎖/麻痺/跑速)、冰❄️(緩速/控場)。每 5 等 1 點，最多 10 階，可專精或混搭。

### 2. 天賦樹（🌳 按鈕）
三向分支，自由分配天賦點：
- **攻擊**：攻擊力、攻速、暴擊、暴擊傷害、多重箭
- **元素**：穿透、寒冰減速、引燃、連鎖閃電
- **生存**：生命、護甲、吸血、再生、移速

### 3. 裝備與詞條（🎒 按鈕，核心差異）
四個裝備欄：武器、護甲、頭盔、飾品。
裝備依稀有度（普通 / 魔法 / 稀有 / 傳奇）擁有不同數量的**詞條**：

- **數值詞條**：攻擊%、攻速%、生命、暴擊、移速、多重箭、穿透、護甲…
- **特效詞條（玩法核心，可疊加）**：
  - ⚡ 連鎖閃電：命中跳擊附近敵人
  - 💥 暴擊爆炸：暴擊引發範圍爆炸
  - 🩸 吸血：傷害轉化為生命
  - ❄️ 冰霜：命中減速
  - 🔥 燃燒：持續傷害
  - 🌵 荊棘：受擊反傷
  - ♻️ 再生：每秒回血
  - 👹 召喚強化：提升召喚物傷害（召喚流）
  - 📏 攻擊範圍：擴大近戰揮砍範圍
  - 🧲 拾取範圍：掉落物與經驗球在範圍內自動吸取
- **經驗球**：擊殺後經驗化為經驗球噴在地上，需拾取才入帳；數量依等級差（越級打怪 10 顆、正常 3~5 顆、打爛低階怪 1 顆）
- **傳奇專屬特效**：風暴之芯、嗜血、玻璃大炮、雙生箭…
- **城鎮商店**：用金幣**強化**已裝備（每級全詞條 +8%）或**賭裝**（隨機部位）

> Build 的樂趣＝湊出特效組合。例：高暴擊率 ＋ 暴擊爆炸 ＋ 連鎖閃電 → 範圍連環爆。

## 武器類型（由裝備的武器決定攻擊方式）

| 武器 | 類型 | 特性 |
|------|------|------|
| 🏹 弓 | 遠程 | 平衡的直線射箭 |
| ⚔️ 雙手劍 | 近戰 | 大範圍扇形揮砍、傷害高、攻速慢 |
| 🗡️ 匕首 | 近戰 | 快速深刺、範圍窄、暴擊高 |
| 🪄 法杖 | 遠程 | 發射追蹤法球 |
| 📖 法書 | 召喚 | 召喚史萊姆自動追擊敵人（上限 3） |

> 「多重箭、穿透」等投射物專屬詞條只會出現在遠程武器上；天賦只保留通用加成。

## 區域與 Boss（主題化 7 關）

翠綠原野(史萊姆) → 幽暗森林(狼) → 炎熱荒漠(火猴) → 龍宮(蝦兵蟹將) → 冰晶宮殿(冰晶) → 風暴高原(妖鳥) → 黑暗深淵(石頭怪)。
以傳送門連接、需達等級解鎖；怪物有各自外觀，每區 Boss 有專屬招式組合與 emoji 外觀。

每區有一座**六芒星召喚祭壇**：站進圈內讓進度跑滿，閃紅延遲 2 秒後 Boss 降臨。
Boss 會停下**閃紅蓄力**後從**專屬大招池**隨機施放（隕石雨、十字光束、跳躍砸地、全場掃描、超大扇形撕咬…）。

## 檔案結構

```
index.html       # 入口：HUD、面板、Canvas
style.css        # 全部樣式
js/data.js       # 靜態資料：詞條、裝備、敵人、區域、天賦
js/systems.js    # 存檔、裝備生成、數值聚合、戰鬥特效、掉落
js/ui.js         # 背包 / 天賦 / 道具彈窗 / HUD
js/game.js       # 主迴圈、輸入、相機、敵人 AI、傳送門
```
