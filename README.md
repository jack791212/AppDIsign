# 🏹 弓箭傳說 Lite

一個受《弓箭傳說 (Archero)》啟發的手機**直版（縱向）**網頁遊戲，純 HTML5 + Canvas，
不需安裝任何東西，手機瀏覽器打開就能玩。

## 玩法

- **拖曳螢幕任意位置**移動角色（虛擬搖桿，可手機單手操作）
- **停下不動**時角色會自動朝最近的敵人射箭
- 邊走邊閃避敵人和子彈
- 清光一關所有敵人即可進入下一關，難度逐關提升
- 擊殺敵人累積經驗，**升級時從三項能力中選一個**：
  攻擊、攻速、生命、移速、多重箭、穿透、治療

敵人類型：
- 🔴 紅色：近戰，會衝向你撞擊
- 🔵 藍色：遠程，會保持距離並射擊

## 在手機上測試的方法

### 方法 A：GitHub Pages（推薦，能在真手機上玩）
本專案已設定自動部署。推送到 `main` 後，到 repo 的
**Settings → Pages**，把 Source 設成「GitHub Actions」即可。
部署完成後會得到網址：

```
https://jack791212.github.io/appdisign/
```

手機瀏覽器打開該網址即可直接遊玩。

### 方法 B：本機區域網路測試
在電腦的專案資料夾執行：

```bash
python3 -m http.server 8000
```

手機連同一個 Wi-Fi，瀏覽器打開 `http://<電腦IP>:8000`。

### 方法 C：直接開檔
電腦上直接用瀏覽器開啟 `index.html` 也能玩（滑鼠拖曳操作）。
