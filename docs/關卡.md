# 關卡（區域）

定義於 `js/data.js` 的 `G.AREAS`。正方形地圖；`reqLevel` 為傳送門解鎖等級；
`elem` 為該區敵人攻擊附帶的元素（對玩家造成 火=燃燒 / 冰=緩速 / 雷=麻痺）。
每個戰鬥區深處有**六芒星祭壇**，站入填滿進度後延遲 2 秒召喚 Boss。

| 區域 | 等級 | 解鎖 | 元素 | 怪物 | Boss |
|------|------|------|------|------|------|
| 城鎮 town | - | - | - | （安全區，鐵匠🔨／女神🧚） | - |
| 翠綠原野 plains | 1 | - | - | 史萊姆、小史萊姆 | 史萊姆王 |
| 幽暗森林 forest | 5 | Lv5 | - | 野狼、史萊姆、爆裂蟲 | 狼王 |
| 炎熱荒漠 desert | 10 | Lv10 | 火 | 火猴、野狼、爆裂蟲 | 火猴王 |
| 龍宮 dragon | 16 | Lv16 | - | 蝦兵、蟹將、冰晶 | 龍王 |
| 冰晶宮殿 ice | 23 | Lv23 | 冰 | 冰晶、蟹將、野狼 | 冰雪女王 |
| 風暴高原 storm | 31 | Lv31 | 雷 | 妖鳥、冰晶、爆裂蟲 | 巨大鳥妖 |
| 黑暗深淵 abyss | 40 | Lv40 | 雷 | 石頭怪、妖鳥、爆裂蟲、火猴 | 深淵巨獸 |

- 怪物 HP 縮放 `×(1+lvl×0.15)`；傷害 `×(1+lvl×0.16)`；經驗 `×(1+lvl×0.18)`；金幣：`gold * (1 + area.level*0.22)`。
- Boss HP 縮放：`hp * (1 + area.level*0.1)`；Boss `tier = round(area.level/7)`，越高大招持續時間/數量越多。
- 想新增關卡：在 `G.AREAS` 加一個區域，串好 `portals`（前往/返回）與 `boss`、`enemies`、`bossAt`。

> 目前 7 個戰鬥區。若要擴充到 10 關，依此格式再加 3 區並設定主題怪物/Boss 即可。
