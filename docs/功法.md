# 功法（氣功）系統

定義於 `js/data.js` 的 `G.QIGONG`，三條平行：**火 🔥 / 雷 ⚡ / 冰 ❄️**。

## 規則
- 每 **5 等**獲得 **1 點**功法點，最多 **10 點（10 階）**（Lv50）。
- 每點讓**某一條往下推進一階**（必須由上往下，不能跳階、同節點不重複）。
- 可自由分配：專精一條、或混搭多條形成不同 build。
- 存檔：`save.qi = { fire, lightning, ice }`（各條已推進的深度）。
- 點數函式（js/systems.js）：`G.qiTotal()`、`G.qiSpent()`、`G.qiAvail()`、`G.qiAdvance(col)`。
- 結算：`computeStats()` 會把各條深度內的節點全部累加（stat 或 proc）。

## 主題
- **火**：傷害／燃燒（burn、dmgPct、critboom、critDmgPct）
- **雷**：連鎖／麻痺／跑速（chain、movePct、atkSpdPct、paraOnHit）
- **冰**：緩速／控場／防禦（frost、armorFlat、hp、freezeChance）
