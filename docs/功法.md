# 功法（氣功）系統

定義於 `js/data.js` 的 `G.QIGONG`，三條平行：**火 🔥 / 雷 ⚡ / 冰 ❄️**。

## 規則
- 共 **10 排**，每 **5 等**開放下一排（Lv50 全開）。
- **每排在 火/雷/冰 三者擇一**；選定後**該排另外兩個永久鎖死**，無法再選。
- 下一排要等再 5 級才開放（Lv5 選第1排、Lv10 選第2排…）。
- 因此最終是一條「每排選一個元素」的路線，可純走一系或混搭。
- 存檔：`save.qi = { picks: [] }`，`picks[排]='fire'|'lightning'|'ice'`。
- 函式（js/systems.js）：`G.qiTotal()`(可選排數)、`G.qiPickRow()`(下一排)、`G.qiAvail()`、`G.qiPick(col)`。
- 結算：`computeStats()` 套用每個已選排所對應元素該排的節點。

## 主題
- **火**：傷害／燃燒（burn、dmgPct、critboom、critDmgPct）
- **雷**：連鎖／麻痺／跑速（chain、movePct、atkSpdPct、paraOnHit）
- **冰**：緩速／控場／防禦（frost、armorFlat、hp、freezeChance）
