# 系統

## 等級 / 經驗
- 升級需求：`G.xpForLevel(lv)=floor(12×lv^1.4)+15`（js/systems.js）。
- 每級 +1 天賦點；每 5 級 +1 功法點。
- 等級提升基礎攻擊與生命。

## 經驗球（🧋珍奶／🥟小籠包）
- 擊殺後經驗化為經驗球噴在地上，需拾取才入帳。
- 數量依等級差（`G.xpOrbCount`）：怪比你高 ≥5 級→10 顆；差距內→3~5 顆；你高 ≥5 級→1 顆。

## 金幣（🪙）
- 擊殺噴出金幣，需拾取入帳。

## 拾取
- 基礎拾取範圍 28（約半個身位，角色身側白圈示意），需走近才吸取。
- 可由詞條「拾取範圍」與天賦「貪婪」增加；🧲 磁鐵可一次吸全圖。

## 關卡流程與引導
- 每區需先**擊殺 20 隻**小怪才能啟動祭壇（頂部顯示進度）。
- 站入祭壇填滿 → 召喚 Boss（頂部變「擊敗 Boss」）。
- 擊敗 Boss → 頂部顯示「已通關」，並出現**指向下一關傳送門的箭頭**。
- 障礙物：戰鬥區有石塊阻擋移動。
- 寶箱 🎁：連殺達 20 時刷新一個（並重置連殺），開啟有**更高機率**出好裝備；右側顯示刷新進度。

## 連續擊殺
- 3 秒內持續擊殺累積連殺數；中斷重置。
- 文字僅在擊殺瞬間震動，連殺越多震幅越大。

## 異常狀態（敵人對玩家）
- 🔥 火＝燃燒（持續傷害）、❄️ 冰＝緩速、⚡ 雷＝麻痺（移動偶爾頓一下）。
- 由區域 `elem` 決定；角色頭上顯示狀態文字。
- 玩家對敵人：功法雷線可麻痺、冰線可定身（停頓）。

## Boss 真實傷害
- 任何 **Boss 攻擊**（彈幕、讀條、光波、接觸）除基底傷害外，**額外附帶 10% 最大生命的真實傷害**（無視護甲），確保再肉再強也必須閃避。一般小怪不附帶。

## 無盡挑戰 / 祝福 Run（深淵之門，Hades 式）
- 城鎮傳送門（需 Lv6）進入即開始一場 **Run**。每間房是獨立競技場。
- 房間流程：進房 → **清光所有敵人** → 出現 **2 道門**（門上顯示下一房獎勵圖示）→ **走進一道門選擇路線**。
- 門的獎勵：🙏祝福 / ❤️回血 / 💰金幣 / ⚔️精英裝備。
- **祝福（Boon）**：走進祝福門時**三選一**（`G.BLESSINGS`，6 位神：炎/雷/冰/劍/風/生命）；**可疊加、離開深淵或死亡歸零**（Run 專用 build，疊在永久裝備/功法之上）。
- 每 **6 房**一個**守關者 Boss**（輪替 7 個 Boss，依房數縮放）。
- 怪物依房數提升等級（`區域等級 + 房×3`）；記錄**最高房數** `save.maxFloor`。
- 祝福資料在 `G.BLESSINGS` / `G.ALL_BOONS`；Run 狀態 `G.run`（startRun/endRun/addBoon）。

### 進階門（Hades 式深化）
門的獎勵池依情境動態組成，除基本 🙏祝福 / ❤️回血 / 💰金幣 / ⚔️精英裝備 外，還可能出現：
- **🌀 混沌門**：獲得**雙倍效果**的強力祝福，但背負 **2 房的詛咒**（傷害 −25%）；詛咒每過一房遞減，歸零自動解除。
- **⬆️ 祝福升級（pom）**：三選一，把一個**已擁有的祝福**強化一級（效果 ×等級，`G.run.boonLv`）。需已有祝福才會出現。
- **🏪 深淵商店**：花金幣買一次性強化（祝福 / 全滿回復 / 精英裝備），購買後離開商店前進。金幣 ≥60 才會出現。
- **✦ 二重祝福（Duo）**：湊齊**兩位指定神**的祝福後，祝福門有機會出現該神組合的強力 combo（`G.DUOS`，如雷炎爆發、絕對凍刃、生命疾風、冰雷風暴）。

## 精英怪
- 一般約 9%、深淵約 16% 機率生成精英怪（金色光環、體型更大）。
- 血量 ×3.6、傷害 ×1.45、經驗/金幣大幅提高，**保證掉落 1~2 件裝備**。
- 隨機詞綴：`fast`(更快)、`tough`(更肉)、`explode`(死亡範圍爆炸)。

## 深淵結晶 + 深淵之鏡（永久 Meta）
- 每次深淵 Run 結束（離開或死亡）依到達房數獲得 💎**深淵結晶**（`約 房數×1.5`）。
- 城鎮女神對話 →「🔮 深淵之鏡」花結晶點**永久被動**（攻擊/生命/護甲/移速/暴擊/暴傷/拾取），always 生效。定義 `G.MIRROR`、`save.mirror`。

## 套裝
- 稀有以上裝備有機率成為套裝件（`item.setId`，4 套：狂戰/寒霜/活力/雷暴）。
- 同套 **2 件**、**4 件**各有加成（`G.SETS` 的 b2/b4）。裝備欄湊齊四部位＝4 件。
- 道具視窗顯示套裝加成與已裝件數。

## 每日任務 + 成就（女神 →「🎯 目標」）
- **每日**：每天隨機 3 則（擊殺/Boss/深淵/金幣/精英），完成給結晶（`G.DAILY_POOL`、`save.daily`）。
- **成就**：一次性里程碑（等級/Boss/深淵層/強化/傳奇/金幣），達成給結晶（`G.ACHIEVEMENTS`、`save.achievements`）。
- 事件由 `G.trackEvent(type,amt)` 追蹤；`G.checkAchievements()` 判定。

## 死亡
- 死亡損失 20% 金幣；等級/裝備/功法/戰利品保留。回城復活。

## 長期目標
- Boss 首殺：額外金幣 + 保證傳奇；每擊敗一種 Boss 永久 +3% 攻擊、+20 生命。

## 操作
- 手機：下方 15% 固定虛擬搖桿（實心區）。
- 電腦：WASD/方向鍵移動；B 背包、C 天賦、T 回城；開始畫面可切換操作模式。
- 自動攻擊（移動中也會攻擊）。主動技能目前關閉（`SKILLS_ENABLED=false`）。

## 城鎮
- 🔨 鐵匠：強化 / 賭裝（對話開啟）。🧚 女神：回復生命。
- 回城後出現回歸傳送門可傳回原區域。
- 首次進入有開場劇情（村長贈弓）。
